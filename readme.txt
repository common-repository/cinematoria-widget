Contributor: Cinematoria
Tags: movie, trailer, poster
Requires at least: 2.8.0
Tested up to: 3.4.2
Stable tag: trunk

== Description ==
Cinematoria Widget displays new, top or coming soon movies.

== Installation ==
1. Extract cinematoria-widget.zip to your hard drive
2. Copy cinematoria-widget into /wp-content/plugins/ directory of your wordpress site
3. Activate Cinematoria widget through the Plugins menu in wordpress admin panel
4. Use the Widget panel to place and configure Cinematoria widget in a sidebar
